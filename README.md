# Telegram bot для автопостинга

Этот проект создаёт Telegram-бота, который:

- принимает текст от владельца и публикует его по заданному интервалу в выбранный чат;
- умеет публиковать в конкретную тему форума;
- может отправлять пост с inline-кнопками;
- продолжает постинг, пока вы не поставите задачу на паузу или не удалите её;
- позволяет менять текст сообщения без пересоздания задачи;
- даёт доступ к управлению только разрешённым `user_id`;
- позволяет из панели владельца добавить доступ для дополнительного аккаунта.

## Что умеет бот

- `/addpost <chat_id> | <интервал> | <текст>`: создать автопост по интервалу.
- `/addpost <chat_id> | <интервал> | <topic_id или -> | <кнопки или -> | <текст>`: создать автопост в тему и с кнопками.
- `/listposts`: посмотреть все задачи.
- `/editpost <id> | <новый текст>`: поменять текст.
- `/editpostfull <id> | <интервал> | <topic_id или -> | <кнопки или -> | <новый текст>`: поменять интервал, тему, кнопки и текст.
- `/pausepost <id>`: остановить публикации.
- `/resumepost <id>`: снова включить публикации.
- `/deletepost <id>`: удалить задачу.
- `/chatid`: показать ID текущего чата.
- `/topicid`: показать ID текущего чата и темы.

## Доступ к боту

- Главный владелец задаётся через `OWNER_ID` в `.env`.
- Только этот аккаунт и выданные им дополнительные `user_id` могут управлять ботом.
- Дополнительный доступ можно выдать и убрать прямо из кнопочной панели `Доступы`.

## Запуск

1. Установите Python 3.11+.
2. Создайте виртуальное окружение.
3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Создайте `.env` на основе `.env.example`.
5. Укажите:

- `BOT_TOKEN`: токен вашего бота из BotFather.
- `OWNER_ID`: ваш главный Telegram user id.
- `DEFAULT_TIMEZONE`: например `Europe/Moscow`.

6. Запустите бота:

```bash
python main.py
```

## Как использовать автопостинг

1. Добавьте бота в нужный чат или канал.
2. Дайте ему право писать сообщения.
3. Узнайте `chat_id` чата.
4. Напишите боту команду, например:

```text
/addpost -1001234567890 | 12h | Доброе утро, друзья
```

После этого бот будет отправлять это сообщение каждые `12` часов.

Пример для темы и кнопок:

```text
/addpost -1001234567890 | 1h | 55 | Блог=https://site.ru; Услуги=https://site.ru/services | СПИСОК НАШИХ ПРОЕКТОВ
```

Если нужно изменить текст, используйте:

```text
/editpost 1 | Новый текст для автопостинга
```

## Важно

- Для каналов бот должен быть администратором.
- Расписание хранится в SQLite, поэтому задачи сохраняются после перезапуска.
